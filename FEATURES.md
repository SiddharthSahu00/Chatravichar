# NoteForge v0.1.0

## Core Features
- Structured note generation from any learning material
- AI-powered revision sheet creation
- Personalized study plan generation
- Quiz and self-assessment creation
- Multi-turn conversation memory
- Faithful responses grounded in source material

## How It Works
NoteForge uses LangGraph for agent orchestration, ChromaDB for vector storage, and Groq LLM for natural language generation.
