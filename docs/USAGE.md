# NoteForge User Guide

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your GROQ API key
4. Run: `streamlit run app.py`

## Features

### Structured Notes
Ask: "Generate notes on [topic]"

### Revision Sheets
Ask: "Create a revision sheet for [topic]"

### Study Plans
Ask: "Create a study plan for [subject]"

### Quizzes
Ask: "Generate a quiz on [topic]"

## Conversation
- Start a new conversation using the sidebar button
- The system remembers your name across messages
- Faithfulness score shown for each response
